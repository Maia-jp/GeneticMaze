    Como funciona ?

    Algorítimos Genéticos

    Algorítimos Genéticos usam o conceito de evolução da biologia para resolução de problemas da computação! Existem diversos problemas onde esses algoritmos podem ser usados como o famoso “Problema do cacheiro viajante”. No caso desse programa o objetivo é percorrer um labirinto.

    Para conseguir resolver um dado problema esses algorítimos simulam o processo de evolução que ocorre em diversas gerações. Em suma cada geração recebe um conjunto de instruções, ao final do seu ciclo de vida os melhores indivíduos (escolhidos de acordo com um critério) são escolhidos para juntar seus “genes” (instruções) para a próxima geração.

    Os Indivíduos

    Nesse programa as ações são feitas pequenos indivíduos (“Unit”) que são representados por pontos dentro do labirinto..

    Cada ponto possui um código genético único e pode (ou não) passar seus genes para próxima geração. Cada um desses pontos “vive” por uma geração.

    Cada individuo pertence a uma das 3 categorias. Pontos vermelhos são indivíduos normais; roxos, sofreram mutação a partir da geração anterior. Amarelos são os melhores indivíduos até agora

    Código Genético

    Quando são criadas, cada individuo/unidade recebe um código genético. Esse código nada mais é do que uma série de genes que representam instruções.

    No nosso programa essas instruções são números de 1 a 4 que representam possíveis ações dentro do tabuleiro. 1-Direita; 2-Desce; 3-Esquerda; 4-Sobe;

    Crossover

    No final da "vida" desses indivíduos os melhores são escolhidos para fazer crossover.

    Crossover é o processo de juntar duas sequencias genéticas (conjunto de instruções) em um novo individuo que pertencera a próxima geração.

    Exemplo, Se temos os seguintes códigos: A-[1,2,3,4] e B-[1,1,1,1], caso seja feito um crossover um dos possíveis resultados seria C-[1,1,3,4]. C faria parte da próxima geração.

    As mesmas unidades podem fazer crossover diversas vezes.

    Geração

    O conceito de geração é similar ao de biologia. Uma geração é composta por individuo que são filhos da geração anterior (com exceção da primeira geração).

    Dentro da população que compõe uma geração os melhores indivíduos são selecionados para passar seus genes para próxima geração.

    Fitness

    Fitness é o valor que mede a performance de um individuo na resolução do labirinto. Esse valor define o quão bem um "célula" se saiu

    Atualmente esse valor é calculado pela distância entre o individuo e o final do labirinto.

    Mutação

    Mutação é um processo de suma importância para gerar diversidade. Esse processo evita que o algoritmo fique preso em um máximo/mínimo local.

    A cada geração existe uma pequena possibilidade de um crossover acontecer e todos os genes serem alterados para algo novo.
